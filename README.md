## 💻 Requisitos e instalación

- Windows 10/11 (64-bit) · Android 8+.
- Si Windows SmartScreen avisa: **Más información → Ejecutar de todas formas** (app aún sin firma comercial).
- 3 pasos: descarga → abre → pega tu clave de IA y transmite. **¿No tienes clave?** La app te guía paso a paso para sacar una gratis.

## 🔍 Seguridad y verificación

- Verifica el **SHA-256** publicado en las notas de cada release.
- Recomendado: sube el binario a **VirusTotal** antes de instalar. Un Electron sin firma comercial puede dar un falso positivo aislado; el reporte lo aclara.

## ⚠️ Uso responsable

Transmitir a varias plataformas a la vez, los canales automatizados 24/7 y usar música con

seguidores mínimos para dar RTMP o desincentivan contenido 100% automático) y con derechos
